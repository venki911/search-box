    Search Box¶ ↑

    This project is part of a test for helpjuice. This project can be found in a heroku app called [Search Box](search-box.herokuapp.com/)

    Requirements¶ ↑

    • Ruby 2.3.0

    • Rails 4.2.6

    • postgres 9.4+

    Installation¶ ↑

    git clone git@github.com:marcoT89/search-box.git
cd search-box
bundle install

    Configure the database connections in `config/database.yml`. Than setup the database with:

    rake db:setup

    Than starts the rails server:

    rails s

    See `localhost:3000` to see the app running.
